== Description ==

### Huge-IT Colorbox

Huge-It Colorbox is the most spellbinding plugin in WordPress that implement Lightbox-effect look of the images and videos (when you click on the thumbnail of the image/video it nicely opens and increases in the same window with a beautiful effect). our Colorbox plugin is distinctiveness because of two things: 
first - it starts working immediately after activation, and does not require any additional configuration for images, and the second is that it has many settings and if you wish, you can configure the plugin for any task. The plugin is able to discover not only beautiful images, but also video and other url links.  Among wonderful abilities of the plugin you can find:

* Ability to customize it’s form,size or color
* It has 5 nonrecurring views in General Options section and you can always choose the best for you. 
* You will be able to change the position of colorbox and also the style of its’ opening.
* What can be more easy and elegant? Just install try and enjoy

== Installation ==

### First download the ZIP file ,

1. Log in to your website administrator panel.   
2. Go to Plugins page, and add new plugin.    
3. Upload [WordPress colorbox](http://wordpress.org/plugins/colorbox). 
4. Click `Install Now` button.     
5. Then click `Activate Plugin` button. 

Now you can set your colorbox options, images and use our colorbox.
